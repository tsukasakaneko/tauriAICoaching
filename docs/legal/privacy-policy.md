# プライバシーポリシー

CoachMate for VALORANT(以下「本サービス」)における個人情報およびユーザーデータの
取り扱いを定めます。

## 1. 収集する情報

### 1.1 アカウント情報
- メールアドレス、パスワード
- これらは**お客様の PC 内のローカルデータベースに保存**されます。パスワードは
  bcrypt によりハッシュ化して保存され、平文では保存されません。

### 1.2 ライセンス・購入情報(当方サーバーに保存)
- ライセンスキーのハッシュ値(キーそのものは保存しません)
- 購入時のメールアドレス(ライセンスキーの送付・照会のため)
- 端末識別ハッシュ(OS のマシン ID から生成した匿名ハッシュ。キーの不正共有防止のため)
- クレジットの付与・消費履歴(残高管理のため)

### 1.3 決済情報
- 決済は Stripe, Inc. が処理します。**クレジットカード番号が当方のサーバーに送信・保存される
  ことはありません**。Stripe の個人情報の取り扱いは [Stripe のプライバシーポリシー](https://stripe.com/jp/privacy) をご確認ください。

## 2. 録画データの取り扱い(重要)

- 自動録画機能で取得した**動画・フレーム画像は、すべてお客様の PC 内でのみ処理・保存され、
  外部のサーバーへ送信されることはありません**。
- 録画の解析(KDA・ヘッドショット率・ポジション等の計測)も、お客様の PC 内で実行されます。
- 録画ファイルはお客様ご自身でいつでも削除できます。

## 3. AI 分析時に外部へ送信される情報

AI コーチングレポートの生成時、以下の情報が分析のために Anthropic, PBC の API
(クラウド AI 利用時は当方サーバーを経由)へ送信されます。

- ランク帯、使用エージェント、自己評価の選択項目、試合振り返りの入力テキスト
- 録画解析から算出した統計値(KDA・ヘッドショット率・ダメージ・活動エリア・ラウンド勝敗等の数値データ)

**動画・画像そのものが送信されることはありません。** Anthropic API に送信されたデータの
取り扱いは [Anthropic のプライバシーポリシー](https://www.anthropic.com/legal/privacy) に従います。
ローカル AI(Ollama)を選択した場合、分析内容は外部へ送信されません。

## 4. 利用目的

- 本サービスの提供(ライセンス認証・クレジット管理・AI 分析)
- 購入されたライセンスキーの送付、購入に関するお問い合わせ対応
- 不正利用(キーの共有・残高改ざん等)の防止
- サービスの改善(個人を特定しない統計情報として)

## 5. 第三者提供・処理の委託

法令に基づく場合を除き、個人情報を第三者に販売・提供することはありません。
サービス提供のため、以下の事業者に処理を委託しています。

| 委託先 | 目的 |
|--------|------|
| Stripe, Inc. | 決済処理 |
| Anthropic, PBC | AI 分析(第3項に記載の内容のみ) |
| Twilio Inc.(SendGrid) | ライセンスキーのメール送付 |
| Render Services, Inc. | サーバーホスティング |

## 6. 保存期間

- ライセンス・クレジット情報: サービス提供に必要な期間
- アカウント情報(ローカル): アプリ内の「アカウント削除」でいつでも完全に削除できます

## 7. 開示・訂正・削除の請求

ご本人からの個人情報の開示・訂正・削除のご請求には、本人確認のうえ遅滞なく対応します。
下記の窓口までご連絡ください。

## 8. 改定

本ポリシーを改定する場合は、アプリ内または販売ページで告知します。
